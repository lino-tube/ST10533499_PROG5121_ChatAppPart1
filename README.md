## Features
### Username Validation
  We checking if the username meets the required rules,
  it must contain an underscore and must be no more than 5 characters in length.
  
### Password Validation
 Checking if the password meets the complexity rules.
 It must contain a special character, a capital letter, a number and must be at least 8 characters long.

### Cell Phone Number Validation
 Checking if the cell phone number has an international South African code(+27),
 then it must follow with 9 charaters making it 12 characters in total.
 If the cell phone number is incorrectly(false) entered, the IDE will display an error message.
