## Features
### Username Validation
  We use booleans to check if the username is true or false.
  We check if the username contains an underscore and must be no more than 5 characters in length.
  
### Password Validation
 Checking if the password meets the complexity rules.
 It must contain a special character, a capital letter, a number and must be at least 8 characters long.
 We test if the password is true to the complexity rules or false.

### Cell Phone Number Validation
 Checking if the cell phone number has an international South African code(+27).
 Then follows with 9 numbers making it 12 numbers in total.
 If the cell phone number is incorrectly(false) entered, the IDE will display an error message and will ask the user to try again.
